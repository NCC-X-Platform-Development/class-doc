## Widget とは

- Flutter の肝
- UI を構成する各要素を Widget と呼ぶ
- Flutter では、 Widget というものをブロックのように組み合わせて UI を作る
- Widget には様々なプロパティが用意されている
  - このプロパティに値を指定して、色や大きさなどを表現する
