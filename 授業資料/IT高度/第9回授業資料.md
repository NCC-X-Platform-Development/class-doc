### 画面遷移について

以下コードで画面遷移ができます。

```dart
// push
Navigator.of(context).push(
  MaterialPageRoute(
    builder: (context) {
      // `XXPage` には遷移先の画面のクラス名を入れる
      return XXPage();
    },
  ),
);

// pop
Navigator.of(context).pop();
```
