# 授業の進め方

- 全体向けのチャットでも個別チャットでも挙手しての口頭でもお好きな方法でどうぞ👍️

- 間違いの指摘や、追加してほしい内容の要望などがあれば Issue を立ててください
  - どんな些細なものでもうぇるかむ🎊

- 良い使い方見つけたら教えてください🙏
